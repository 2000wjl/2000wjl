  1. 等价类划分法
  2. 边界值分析法
  3. 因果图法

12存在缺陷，未考虑输入输出、输入输入、输出输出关系。故引入因果图

因果图（Cause and Effect Diagram）

​ 因：输入

​ 果：输出

因与因(E I O R)

  1. Exclusive 排外的

    C1 C2 C3不可同时成立

  2. InClusive 包含

    C1 C2 C2至少一个为1(True) 不可全部为0

  3. Only

    C1 C2必须有,且仅有一个为1。

  4. Require

    C1为1时，C2必定为1。反之，C2为1,C1不必为1。

    Eg:a>90 b>60 则a-1–>b-1

因与果(恒等-相等、与、或、非)

  1. 相等

    if(C) then Ef

  2. 与

    if(C1&&C2&&C3) then Ef

  3. 或

    if(C1||C2||C3) then Ef

  4. 非

    if(!C) then Ef

果与果（M约束）

  1. Masking

    Ef1为1时，Ef2为0。反之，不可推。
